Getting Started

Introduction to Vortex

Vortex is a distributed AI infrastructure layer built to automate model scaling. It abstracts the complexity of GPU provisioning, allowing you to focus on building rather than DevOps.

Vortex 2.0 now supports automatic weight-sharding for models exceeding 70B parameters.

Key Features

  • Zero-Config Scaling: Your models grow as your traffic does.
  • Multi-Region Edge: Deploy closer to your users for < 50ms latency.
  • Hot-Swapping: Update model versions without downtime.

Installation

To get started, install the Vortex CLI using your preferred package manager:

$ npm install -g vortex-ai
